Frontend Development Showcase

In 2016, I wanted to learn programming. So for 3 weeks I went on a coding spree and went from being basically a beginner to building a bunch of really cool front end web projects. All the following projects were built within 21 days from September 21, 2016 to October 12, 2016. This is a little journey through my process. I learned everything here by following the curriculum at Free Code Camp.

All the code can be found in the GitHub repo for this site.

Pomodoro Clock

This is a little clock utility. When you are procrastinating, starting is the hardest part. Once you are working, most people get into the flow of things and don't mind work so much. But starting is killer. So you trick yourself into starting by promising to only work for a short amount of time (say 25 minutes), then giving yourself a break. Breaking work into 25 minute segments is called the Pomodoro Technique.

Game of Life

This is a simulation called Conway's Game of Life after the mathematician who invented it.

The cells flip on or off, (alive or dead) based off of how many cells around them are on. It is meant to simulate a bacterial colony. If a cell has too many neighboring cells that are also alive, it dies due to overpopulation. But if a cell has too few neighbors, it dies as it can't reproduce.

Try making a glider for good fun. Press the pause button then click on the cells to toggle them on and off.

This project really shows the strength of React. Having each cell manage it's own state based on event calls really simplifies the code.


Dungeon Crawler

This is a rouge like dungeon crawler role playing game. Use the arrows on the keyboard to move around.

Red squares are enemies, yellow squares are better weapons, and green squares give you more health. Bump into an enemy to attack it. Make your way through the maze to find the final boss in the flashing room.

At almost 500 lines of React JavaScript goodness, this project took some serious work.
